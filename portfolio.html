16bit CPU in HDL

A 16-bit CPU engineered using a nand2tetris hardware simulator, capable of running intricate 16-bit machine code. This processor was then joined to a 23k RAM, 26k ROM, with a keyboard input and screen output. This project was a testimate to my ability to understand many new conepts and translate them into concrete working simulations.
